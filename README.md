# ⚡ Aelo

Plataforma de comunicação por voz e texto em tempo real.

## Como instalar

Baixe o instalador mais recente na aba [Releases](https://github.com/Lucastecnology/aelo/releases).

## Desenvolvimento

```bash
npm install
npm run electron:dev
```

## Tecnologias

- React + Vite
- Electron
- Supabase (Realtime)
- PeerJS (Voice)
- Framer Motion
